# Student Academic Performance Analytics Using SQL

## Project Overview

This project analyzes student academic performance using SQL Server and SQL Server Management Studio (SSMS). The objective is to evaluate student success, engagement, academic risk, module performance, and demographic patterns using data-driven analysis.

## Dataset

Student Academic Performance Dataset

## Tools Used

* SQL Server
* SQL Server Management Studio (SSMS)

## Skills Demonstrated

* KPI Analysis
* Student Segmentation
* Academic Risk Analysis
* CASE Statements
* Aggregations
* Common Table Expressions (CTEs)
* Window Functions
* Univariate Analysis
* Bivariate Analysis
* Multivariate Analysis

## Key KPIs

* Overall Pass Rate
* Average Student Score
* Student Engagement Rate
* Academic Excellence Rate
* High-Risk Student Ratio
* Student Performance Index

## SQL Techniques Used

* GROUP BY
* HAVING
* CASE WHEN
* Aggregate Functions
* CTEs
* Window Functions
* Ranking Functions
* LAG & LEAD
* Running Totals
* Moving Averages

## Key Findings

* Student engagement metrics help identify potentially high-risk students.
* Module-level analysis provides insight into academic performance patterns.
* Student segmentation supports targeted academic interventions.
* Performance metrics can be used to support data-driven academic decision making.

## Recommendations

* Develop early-warning systems for low-engagement students.
* Monitor module-level performance regularly.
* Use engagement metrics for academic support planning.
* Apply student segmentation for personalized academic advising.

## Conclusion

This project demonstrates the use of SQL for educational analytics, KPI development, segmentation, academic risk analysis, and business-oriented reporting.
